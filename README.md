## Timestamp to string.

A small util for converting a timestamp to a string.
- "XX seconds"
- "XX minutes"
- "XX hours"
- "XX weeks"
- "XX months"
- "XX years"

## Installation

```
npm install
```

## Run tests

```
npm test
```

Code coverage can be found in the coverage folder which is created once running the tests.

## Usage

```
import timestampToString from '../src/time';

timestampToString(milliseconds)
```
